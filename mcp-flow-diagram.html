📦 Context Builder
1. Semantic search → top N docs
2. Fetch full content (excl. embedding)
3. Apply token budget (default: 5000)
4. Smart truncation: section → paragraph → sentence
5. Assemble structured context block
🧮 Embedding Generator
Hash-based deterministic (512-dim)
N-gram hashing for semantic signal
Production path: OpenAI text-embedding-3-small + pgvector
Cosine similarity for vector matching
📥 Ingestion Pipeline
Scan knowledge/ markdown files
Parse frontmatter: title, category, description, keywords
Auto-detect category from path, slug from filename
Generate embedding → upsert to DB
Supports: full / category / single reindex
Prisma ORM → SQLite (production: PostgreSQL + pgvector)
